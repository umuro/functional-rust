# Example 002: List Operations

**Difficulty:** ⭐
**Category:** Recursion, Lists, Higher-Order Functions
**Concept:** Fundamental list operations (length, sum, append, reverse, map, filter) form the backbone of functional programming. They illustrate how recursion replaces loops, and how higher-order functions abstract patterns.
**OCaml → Rust key insight:** OCaml's `head :: tail` pattern matching on linked lists maps to Rust's `split_first()` on slices, but Rust's Vec/slice model means different performance characteristics.
