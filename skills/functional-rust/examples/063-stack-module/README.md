# Example 063: Stack Module with Signature

**Difficulty:** ⭐⭐
**Category:** Modules
**Concept:** Defining an abstract data type with a module signature (OCaml) or trait (Rust). The implementation is hidden behind an interface, enforcing encapsulation. Both persistent (immutable) and mutable variants are shown in Rust.
**OCaml → Rust insight:** OCaml's `module type` signatures map to Rust's `trait` definitions — both define an interface that hides implementation details and enables polymorphism.
