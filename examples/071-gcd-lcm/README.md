# 071 — GCD and LCM

**Difficulty:** ⭐ Beginner
**Category:** Math/recursion
**Concept:** Euclidean algorithm for greatest common divisor
**Key Insight:** The recursive GCD is nearly identical in both languages. Rust's `Iterator::reduce()` is the equivalent of OCaml's `List.fold_left` for combining a list.
