# 1064: Permutations

**Difficulty:** Intermediate
**Category:** Backtracking
**Concept:** Generate all permutations of a collection using swap-based backtracking, used-flags, or Heap's algorithm
**Key Insight:** Swap-based backtracking generates permutations in-place with O(1) extra space per level; Heap's algorithm is the most efficient iterative approach, minimizing swaps to exactly one per permutation.
