# 1066: All Subsets (Power Set)

**Difficulty:** Intermediate
**Category:** Backtracking / Bit Manipulation
**Concept:** Generate all 2^n subsets using backtracking, bitmasking, or iterative doubling
**Key Insight:** Each element has a binary choice (include/exclude), so subsets map naturally to n-bit integers. The iterative doubling approach builds subsets by taking all existing subsets and creating copies with the new element appended.
