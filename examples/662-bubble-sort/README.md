# Bubble Sort

Simple comparison sort that repeatedly swaps adjacent elements.

## Complexity

- **Time**: O(n²) average/worst, O(n) best (optimized)
- **Space**: O(1)
- **Stable**: Yes

## Variants

1. **Standard**: Basic nested loops
2. **Optimized**: Early termination if no swaps
3. **Cocktail Shaker**: Bidirectional passes

## When to Use

- Educational purposes
- Very small arrays (< 10 elements)
- Nearly sorted data (with optimization)
