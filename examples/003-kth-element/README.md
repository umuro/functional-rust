# Example 003: Kth Element

**Difficulty:** ⭐ Beginner  
**Category:** Lists & Indexing  
**OCaml Source:** 99 Problems #3  

## Problem

Find the k-th element of a list (1-indexed).

## Key Differences

- OCaml: Recursive traversal, O(n)
- Rust: Direct indexing `.get(k)`, O(1)
- OCaml uses 1-indexing by convention
- Rust uses 0-indexing (more common)
