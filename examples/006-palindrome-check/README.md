# Example 006: Palindrome Check

**Difficulty:** ⭐ Beginner  
**Category:** Lists & HOF  
**OCaml Source:** 99 Problems #6  

## Problem

Find out whether a list is a palindrome (reads the same forward and backward).

## Learning Outcomes

- Comparing forward and reverse sequences
- List equality
- Using standard library functions

## Key Differences

- **OCaml:** Reverse comparison
- **Rust:** Iterator reversal + equality check
- Both use Option/Result for safety
