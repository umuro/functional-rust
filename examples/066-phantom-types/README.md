# 066 — Phantom Types

**Difficulty:** ⭐⭐⭐ Advanced
**Category:** Functors and modules
**Concept:** Using phantom type parameters for compile-time unit safety
**Key Insight:** Both languages support phantom types, but Rust requires `PhantomData<T>` to satisfy the compiler's "unused type parameter" rule — making the phantom explicit rather than implicit.
