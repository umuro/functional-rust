# Example 008: Eliminate Duplicates

**Difficulty:** ⭐ Beginner  
**Category:** Lists & HOF  
**OCaml Source:** 99 Problems #8  

## Problem

Eliminate consecutive duplicates of list elements.

## Learning Outcomes

- Stateful iteration
- Deduplication patterns
- Accumulator-based solutions

## Key Differences

- **OCaml:** Accumulator
- **Rust:** .dedup() or custom accumulator
- Both use Option/Result for safety
