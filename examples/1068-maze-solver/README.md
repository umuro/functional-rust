# 1068: Maze Solver

**Difficulty:** Intermediate
**Category:** Backtracking / Graph Search
**Concept:** Find a path through a 2D grid maze using DFS backtracking or BFS shortest path
**Key Insight:** DFS backtracking finds *any* path (not necessarily shortest) by exploring and undoing; BFS guarantees the *shortest* path by exploring level-by-level, using a parent array for reconstruction.
