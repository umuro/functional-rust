# Example 004: List Length

**Difficulty:** ⭐ Beginner  
**Category:** Recursion & Tail Recursion  
**OCaml Source:** 99 Problems #4  

## Problem

Find the length of a list.

## Learning Outcomes

- Tail recursion optimization
- Accumulator pattern
- Iterator-based counting

## Key Differences

- OCaml: Tail-recursive with accumulator
- Rust: O(1) `.len()` for slices, or `.count()` for iterators
- Demonstrates why recursion matters less in Rust
