# Merge Sort

Divide and conquer algorithm with guaranteed O(n log n) performance.

## Complexity
- **Time**: O(n log n) all cases
- **Space**: O(n)
- **Stable**: Yes

## Variants
1. **Top-down**: Recursive division
2. **Bottom-up**: Iterative merging
3. **Natural**: Exploits existing runs

## When to Use
- Need guaranteed O(n log n)
- Stability required
- External sorting (files)
- Linked lists (O(1) space possible)
