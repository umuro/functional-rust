# Functional Rust

OCaml → Rust functional programming examples.
Published at: https://hightechmind.io/rust/

## Structure

Each example in `examples/NNN-name/` contains:
- `README.md` — explanation, difficulty, learning outcomes
- `example.ml` — OCaml implementation
- `example.rs` — Rust implementation with tests
- `COMPARISON.md` — deep side-by-side analysis

## Generation

Site HTML is generated by:
```bash
python3 ~/.openclaw/workspace/scripts/generate-rust-site.py
```

## Deployment

scp -P 65002 -i ~/.ssh/id_ed25519 -o StrictHostKeyChecking=no \
